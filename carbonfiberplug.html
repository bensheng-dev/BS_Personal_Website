Carbon Fiber Plug

Skills: Solidworks Surfacing

September 2025 - Present

The scope of this project is to surface a shell to create the carbon fiber aeroshell. Creating this shell requires me to create indents for all the lights, fairings and other cut-outs; this involves a lot of solidworks surfacing. Here are three targets for this project:

1. Create a mold to build the shell

2. No sizable gaps in CAD model

3. Speed, needs to be completed within one month
